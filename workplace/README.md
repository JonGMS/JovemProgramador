# 🚀 Microprojetos - Curso Jovem Programador

Este repositório reúne pequenos projetos desenvolvidos durante o curso **Jovem Programador**.
O objetivo é praticar conceitos de programação, lógica, estrutura de dados e desenvolvimento de aplicações.

Cada pasta contém um microprojeto independente, criado para exercitar diferentes conceitos aprendidos ao longo do curso.

---

## 🧠 Conceitos Praticados

Entre os conceitos utilizados nos projetos estão:

* Lógica de programação
* Estruturas condicionais
* Estruturas de repetição
* Vetores e listas
* Programação orientada a objetos
* Manipulação de arquivos
* Aplicações em console
* Organização de código

---

## 🛠 Tecnologias Utilizadas

* **C#**
* **.NET**
* **Visual Studio / VS Code**

---

##  Objetivo do Repositório

Este repositório serve para:

* Registrar minha evolução no curso
* Armazenar exercícios e exemplos práticos
* Servir como material de consulta futura
* Demonstrar prática de programação
